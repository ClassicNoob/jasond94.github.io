About Jason

Jason Downing is a Junior at the University of Massachusetts, Lowell, where he studies Computer Science. Jason's hobbies include mountain biking, skiing, PC gaming and coding.

In the past, Jason worked at several retail stores, and as a research assistant for the Engaging Computer Group at UMass Lowell. Jason currently works for Fidelity Investments as a Technical Intern. He worked Summer 2015, and is back at Fidelity Investments for Summer 2016.

Jason spent Summer 2015 at Fidelity writing COBOL, JCL and DB2 / SQL queries for Fidelity's Brokerage system, which runs on IBM Mainframes.

Jason has currently spent Summer 2016 at Fidelity working on Fidelity's Android App. Jason has spent time working on new features for the app. He created several new app screens using XML and Android Studio. Jason then wrote Java classes to create dynamic text, UI logic, and improve the user experience.
